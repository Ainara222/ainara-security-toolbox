# 🛡 Ainara Security Toolbox

Este repositorio es el núcleo de mi proyecto personal enfocado en auditorías de seguridad. Aquí es donde aterrizo y conecto todo lo que voy aprendiendo en mis estudios, utilizando la IA como copiloto para potenciar el análisis y la creación de herramientas de seguridad más eficientes. No es solo un almacén de archivos; es el motor que uso para realizar mis propias auditorías.

El contenido nace de mis apuntes y documentación de clase, los cuales he procesado y transformado a formato Markdown para que sean navegables y prácticos. He filtrado la teoría para quedarme con lo que realmente aporta valor en escenarios reales, organizándolo en tres pilares fundamentales:

Digital Forensics: Investigación de incidentes y recolección de evidencias.

System Hardening: Técnicas de bastionado para blindar sistemas y redes.

Compliance: Gestión de riesgos, normativa y planes de securización estratégica.

Cada sección está enriquecida con imágenes, enlaces externos de referencia y esquemas técnicos que me sirven de apoyo visual durante mis intervenciones. Este repo es, en esencia, mi base de conocimiento viva: un recurso donde recojo lo más relevante de mi formación para aplicarlo directamente en mis proyectos actuales de auditoría de forma ética, profesional y siempre buscando la mejora continua.
